# Genogram Tool

家系図（ジェノグラム）をブラウザ上で作成できるツールです。  
家族関係やメモを直感的に操作でき、完成した図は画像として保存できます。直感的でマウスの動きも少なく済むので、手軽に作成できます。
介護の現場などでお使いください。

## 主な機能
- ノード（人物）の追加・削除・編集  
- 関係線（親子・夫婦など）の描画 
- 同居グループの範囲描画
- ローカル保存と再読み込み（予定）

## 使用技術
HTML / CSS / JavaScript（Vanilla）  

## 使い方
1. `index.html` をブラウザ（Chrome推奨）で開く https://chamasei.github.io/genogram_tool/ 
2. 画面上クリックでノード（人物）を選んで追加し、関係線も同様にクリックから選ぶ  
3. 左上のテキストボタンクリックで注釈テキストボックス挿入が可能（再度クリックで画像挿入モードに切替）
4. テキストはボックス右に出てくるミニツールバーのA＋あるいはA-で文字サイズの変更が可能。ミニツールバーのゴミ箱アイコンクリックでテキストボックス削除。
5. 同居グループボタンクリックでモード切替　画面上クリックで同居グループの範囲設定が可能。
6. スクリーンショットで画像保存 

### 操作のコツ
- ドラッグ：ノード移動

- **プライバシー**：データはローカルで処理され、サーバーへ送信されません
