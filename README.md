# Genogram Tool

家系図（ジェノグラム）をブラウザ上で作成できるツールです。  
家族関係やメモを直感的に操作でき、完成した図は画像として保存できます。直感的でマウスの動きも少なく済むので、手軽に作成できます。
介護の現場などでお使いください。

## 主な機能
- ノード（人物）の追加・削除・編集  
- 関係線（親子・夫婦など）の描画  
- ローカル保存と再読み込み（予定）

## 使用技術
HTML / CSS / JavaScript（Vanilla）  

## 使い方
1. `index.html` をブラウザ（Chrome推奨）で開く  
2. 画面上でノード（人物）を追加して関係線を引く  
3. 右クリック/メニューから画像（PNG）として保存

### 操作のコツ
- ドラッグ：ノード移動

- **プライバシー**：データはローカルで処理され、サーバーへ送信されません

## 開発者向け
